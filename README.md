# Java Console-Based Pacman Game

A Java-based console Pacman game developed as an OOP project 
bringing the classic arcade experience to the terminal.

## Features
- Interactive console gameplay with real-time keyboard controls
- Smooth Pacman movement using W, A, S, D navigation
- Intelligent Ghost AI that dynamically tracks player movement
- Score tracking system with bonus fruit rewards
- Life management with collision detection and restart logic
- Maze-style board with walls, food pellets and special items
- Win and Game Over conditions
- Modular and scalable class-based architecture

## Tech Stack
Java | OOP | Console UI | Data Structures | Game Logic
